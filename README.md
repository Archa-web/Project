# Project
# It is a NLP based project. The beginners or even the experts are sometimes unknown with the commands in linux terminal. We build a solution for that to make the use of the terminal simple. We re-build the terminal of linux os into a user-friendly terminal. In this the user don't need to know the command, they can just simply type in the natural language. For eg:- Instead of typing 'pwd' you can type ' Which is my current directory '. And also this terminal provide a safety layer , which check the danger command and safe command and inform the user before the execution. Another implementation is u can enter the sentance either in malayalam or english. This project is build-on using SBERT, T5-small, llama 3.2, NLLB-200 and XlitEngine. Language used here is python. And also we do a RAG implementation.
